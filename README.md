# 🧮 Simple Calculator

A clean and responsive calculator built using **HTML**, **CSS**, and **JavaScript**. It performs basic arithmetic operations with a modern user interface and keyboard support.

## ✨ Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 📊 Percentage calculations
* ⌨️ Keyboard support
* 📱 Responsive design
* 🎨 Modern and clean UI
